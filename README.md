# Аналитический отчёт для агентства недвижимости

## Основная задача проекта:
помощь агентству недвижимости из Петрозаводска в освоении рынка Санкт-Петербурга и Ленинградской области путем анализа данных о продажах жилья. Мы определим перспективные сегменты недвижимости и выявим лучшие сезоны для запуска рекламных кампаний.

**Используемые инструменты:**
- DBeaver — подключение и первичное знакомство с базой данных;
- SQL-запросы — решение ad hoc задач и формирование выводов;
- DataLens — создание и доработка дашбордов.
- 
### Этапы реализации проекта:
1. Изучение данных. Используя DBeaver, изучаем структуру и содержимое архива данных сервиса Яндекс.Недвижимость. Эти данные включают информацию о жилых объектах, продающихся в Санкт-Петербурге и Ленинградской области.

2. Решение ad hoc задач c помощью SQL-запросов определяем:
- наиболее выгодные сегменты недвижимости исходя из длительности пребывания объявлений на платформе;
- сезонные колебания спроса и предложений на рынке недвижимости региона;
- выводы представляем в форме короткой аналитической записки.

3. Создание и улучшение дашборда, а именно визуализируем полученные данные в DataLens, формируя наглядные графики и диаграммы, полезные для принятия решений заказчиком.

### ✅ Результаты проекта
формирование комплекс рекомендаций по выбору целевого сегмента недвижимости и оптимальному периоду размещения рекламы. Этот анализ повысит шансы агентства на успешный выход на рынок Петербурга и Ленобласти.

## Описание данных используемых при  реализации проекта:

Таблица `advertisement` содержит информацию об объявлениях:

- `rooms` — число комнат;
- `ceiling_height` — высота потолка, в метрах;
- `floors_total` — этажность дома, в котором находится квартира;

Таблица `city` содержит информацию о городах:
- `city_id` — идентификатор населённого пункта (первичный ключ);

Таблица `type` содержит информацию о городах:
- `type_id` — идентификатор типа населённого пункта (первичный ключ);
- `type` — название типа населённого пункта.

### Задача 1: Время активности объявлений
Для эффективного планирования стратегии на рынке недвижимости клиенту необходимо выяснить, какие квартиры в Санкт-Петербурге и Ленинградской области быстрее находят покупателя, а какие дольше остаются на площадке объявлений. Основной критерий — срок нахождения объявления в активной фазе ("время активности").

Необходимо разделить все активные объявления о продаже жилой недвижимости по продолжительности срока публикации на следующие категории:
- `1-30 days` — около месяца;
- `31-90 days` — от одного до трех месяцев;
- `91-180 days` — от трех месяцев до полугода;
- `181+ days` — более шести месяцев;
- `Не снятые` с продажи — категория non-category.

Основные метрики к расчету:
- cреднюю цену за квадратный метр;
- cреднюю жилую площадь объекта;
- cреднее количество комнат;
- наличие балкона.

### Задача 2. Сезонность объявлений

Необходимо выявить сезонные закономерности активности на рынке недвижимости Санкт-Петербурга и Ленинградской области.  Исследование проводится с целью понимания наилучшего времени для начала маркетинговых кампаний и оптимального сезона для выхода на рынок.

Основные метрики и характеристики:
 -  период (месяц) с наибольшей активностью;
 -  проверить согласованность сезонов активной публикации и продажи недвижимости;
 -  оценить влияние сезонности на ключевые параметры недвижимости;
 -  среднюю стоимость квадратного метра.
Среднюю площадь квартир.
